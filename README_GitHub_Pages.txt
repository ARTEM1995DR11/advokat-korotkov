Быстрый запуск сайта на GitHub Pages

1) Зайдите на https://github.com и создайте публичный репозиторий, например: advokat-site
2) Загрузите в репозиторий файл index.html из этого архива.
3) Откройте Settings → Pages:
   - Source: Branch = main
   - Folder = /(root)
   Сохраните.
4) Через 1–2 минуты сайт будет доступен по адресу:
   https://ВАШ_ЛОГИН.github.io/advokat-site/
